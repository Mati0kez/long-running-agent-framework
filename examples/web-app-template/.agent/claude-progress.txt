## Quick Reference

### File Locations
- Feature List: `.agent/feature_list.json`
- Tests: `.agent/tests.json`
- Agent State: `.agent/agent_state.json`
- Progress: `.agent/claude-progress.txt`

### State Machine
| State | Meaning |
|-------|---------|
| continuous | Run sessions indefinitely |
| run_once | Execute single session then pause |
| pause | Wait for human command |
| terminated | Agent stopped |

### Common Commands
```bash
# Check project status
cat .agent/agent_state.json

# View feature progress
cat .agent/feature_list.json | grep '"passes"'

# Read session history
cat .agent/claude-progress.txt

# Start development server
./init.sh
```
